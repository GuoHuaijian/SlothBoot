# Code of Conduct

## Our Pledge

为了让 Sloth Boot 成为一个对所有人都更友好的开源项目，我希望社区交流保持：

- 尊重
- 克制
- 真诚
- 聚焦问题本身

## Our Standards

鼓励的行为：

- 以建设性方式提出问题和建议
- 对代码、设计和文档给出具体反馈
- 尊重不同背景和经验水平的贡献者
- 在讨论中尽量给出上下文和证据

不鼓励的行为：

- 人身攻击
- 阴阳怪气、羞辱式表达
- 无上下文地否定他人工作
- 发布明显无关或骚扰性内容

## Enforcement

如果你发现不合适的行为，可以通过仓库维护者公开联系方式或 Issue 进行反馈。  
我会尽量基于事实和上下文处理问题，并优先维护社区讨论环境。
