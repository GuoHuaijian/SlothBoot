# Support

如果你在使用 Sloth Boot 时遇到问题，可以按下面方式处理：

## 使用问题

优先查看：

- 根目录 [README.md](./README.md)
- 示例工程 [sloth-boot-example-service/README.md](./sloth-boot-example/sloth-boot-example-service/README.md)

## Bug 反馈

如果你确认是 Bug，请提 Issue，并尽量提供：

- 使用的 JDK / Maven 版本
- 具体模块
- 报错堆栈
- 最小复现步骤

## 功能建议

如果你希望新增能力，请说明：

- 使用场景
- 为什么适合进入公共模块
- 是否已有临时实现

## 维护说明

这是一个个人维护的开源项目，回复速度可能不会像商业项目那样稳定，但我会尽量持续整理和改进。
